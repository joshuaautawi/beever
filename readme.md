# How to Running the apps 



## 1. Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

## Setting
create .env file that has :
```bash
PORT=
PASSWORD=
````
and create postgres database with beeaver_db in your local database



## Running the API
```bash
npm run migrate
npm run start
```
